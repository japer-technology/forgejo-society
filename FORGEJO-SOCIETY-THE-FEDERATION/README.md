# Forgejo Society: The Federation

> The root Forgejo Society for and by the creator in the spirit of Star Trek.

This is the first society created by Eric Mourant during the implementation of Forgejo Society.

The path to a Forgejo Society is first to install the Forgejo Intelligence workflow into the repo you want to work in, running the workflow brings Forgejo Intelligence capabilities into your repo. Configure the Forgejo Intelligence then use it to either join a society or create your own society. You may then define society minds and labour, these become available to intelligences.
