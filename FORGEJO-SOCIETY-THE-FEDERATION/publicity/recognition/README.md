# `recognition/`

> Awards, mentions, citations, and inclusions.

This folder holds the canonical record of **recognition** received by
Forgejo Society. Top-level rules and the entry format live in
[`../08-recognition.md`](../08-recognition.md).

---

## What belongs here

- **Recognition entries**, one per item, named
  `recognition-YYYY-MM-DD-short-handle.md`.
- **An index file** (`index.md`), added once the first entry is
  logged, grouped by category (awards, citations, inclusions,
  notable mentions).

## What does not belong here

- Self-nominations or self-curated lists.
- Generic aggregator entries.
- Social-media reactions (those, where worth keeping, are handled in
  [`../coverage/`](../coverage/README.md)).

## Conventions

- **Restraint.** Recognition is recorded for the historical record,
  not used as a billboard. Use of recognition in promotional copy is
  governed by
  [`../../FORGEJO-SOCIETY-PROMOTION/08-style-guide.md`](../../FORGEJO-SOCIETY-PROMOTION/08-style-guide.md).
- **Citable.** Each entry includes a stable URL or DOI where one
  exists.
- **Honest framing.** The "Why it matters" section names what the
  recognition does *and does not* signify for the project.

## Status

Scaffold. The first recognition entry is added when the first
external recognition arrives.
