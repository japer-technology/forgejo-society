# `talks/`

> Talk abstracts, slide outlines, demo scripts.

This folder holds the **canonical materials for talks** about Forgejo
Society — at conferences, meetups, podcasts, and federated peer
gatherings. Voice and terminology are governed by
[`../08-style-guide.md`](../08-style-guide.md).

---

## What belongs here

- **Abstracts** — the approved short and long abstracts submitted to
  programme committees.
- **Outlines** — slide-level outlines (text only) that other speakers
  can reuse and adapt.
- **Demo scripts** — step-by-step scripts for live demos of the forge
  acting as a mind, with explicit reset and fail-safe steps.
- **Q&A notes** — anticipated audience questions and approved answers,
  cross-referenced with `../07-press-kit.md` FAQ.

## What does not belong here

- Rendered slide decks in proprietary binary formats (link to them
  from here, host elsewhere).
- Speaker-specific delivery notes that are not meant to be reused.

## File layout (planned)

- `abstract-NN-short-handle.md` — one file per talk.
- `outline-NN-short-handle.md` — companion outline.
- `demo-NN-short-handle.md` — companion demo script.

A simple index table at the bottom of this README is maintained as
talks are added.
