# `web/`

> Landing-page copy, README badges, link cards.

This folder holds **canonical web-facing copy** for Forgejo Society.
The visual identity that governs how this copy is rendered lives in
[`../06-visual-identity.md`](../06-visual-identity.md); voice rules
live in [`../08-style-guide.md`](../08-style-guide.md).

---

## What belongs here

- **Landing-page copy** — hero, sub-hero, sections, calls to action.
- **README badges** — the canonical set of badges used in the root
  `README.md` and pillar READMEs, with their meanings.
- **Link cards** — Open Graph / link-preview copy and image
  references.
- **Navigation labels** — the canonical names used in site navigation,
  aligned with pillar names.

## What does not belong here

- Templates, themes, or static-site generator code (those belong in a
  separate web repository or surface).
- Marketing campaigns or social posts (use `../05-campaigns.md` and
  `../social/`).

## File layout (planned)

- `landing.md` — landing-page copy.
- `badges.md` — canonical badge set.
- `link-cards.md` — Open Graph / link-preview copy.
- `navigation.md` — site navigation labels.

## Status

Scaffold. Files are added as web materials are approved.
