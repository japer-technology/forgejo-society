# `press/`

> Press releases, media inquiries, boilerplate.

This folder holds the **press-facing materials** for Forgejo Society.
Top-level structure and rules live in
[`../07-press-kit.md`](../07-press-kit.md); voice rules live in
[`../08-style-guide.md`](../08-style-guide.md).

---

## What belongs here

- **Press releases** — one file per release, named
  `release-YYYY-MM-DD-short-handle.md`.
- **Boilerplate** — the canonical project boilerplate paragraph (also
  referenced from `../07-press-kit.md`).
- **Fact sheet** — the canonical, versioned fact sheet.
- **Quotes** — approved quotations attributable to project
  maintainers.
- **Contact protocol** — how external parties reach the project, who
  responds, and the expected turnaround.
- **Project poster** — the canonical single-page description of the
  project, suitable for printing or rendering as a one-page PDF; see
  [`poster.md`](poster.md).

## What does not belong here

- Speculative or unapproved statements.
- Embargoed material that has not been reviewed under
  `THE-SOCIETY-OF-REPO/01-governance/`.
- Personal contact details of contributors.

## File layout (planned)

- `boilerplate.md`
- `fact-sheet.md`
- `quotes.md`
- `contact.md`
- `poster.md`
- `release-YYYY-MM-DD-*.md`

## Status

Scaffold. Files are added as press materials are approved.
