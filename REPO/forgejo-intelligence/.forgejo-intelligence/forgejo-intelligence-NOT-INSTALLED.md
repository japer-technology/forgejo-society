# .forgejo-intelligence 🦞 Installation Flag

### Indicates if .forgejo-intelligence is currently installed correctly

<p align="center">
  <picture>
    <img src="https://raw.githubusercontent.com/japer-technology/forgejo-intelligence/main/.forgejo-intelligence/logo.png" alt="Issue Intelligence" width="500">
  </picture>
</p>

The file `.github/workflows/forgejo-intelligence-INSTALLER.yml` determines this flag.

The flag is the renaming of the forgejo-intelligence-NOT-INSTALLED.md/forgejo-intelligence-INSTALLED.md files depending on status.
