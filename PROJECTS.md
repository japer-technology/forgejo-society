# Projects

This file lists the project pillars that make up **Forgejo Society**. Each
pillar is a top-level directory in this repository with its own `README.md`
as the entry point. The pillars are listed in reading order: start with the
introduction, then the runnable instance, then planning, setup, and the
federation that runs on owned hardware.

For the project's overall framing and the five quiet reversals, see
[README.md](README.md). For the binding compliance posture, see
[WARNING.md](WARNING.md).

---

## Pillars

| Pillar | Purpose |
| --- | --- |
| [FORGEJO-SOCIETY-INTRODUCTION/](FORGEJO-SOCIETY-INTRODUCTION/README.md) | The on-ramp to the project. Essays, reading paths, and the canonical reference texts: [Society of Repo](FORGEJO-SOCIETY-INTRODUCTION/THE-SOCIETY-OF-REPO/), [Society of Mind](FORGEJO-SOCIETY-INTRODUCTION/THE-SOCIETY-OF-MIND/), and [Repo is the Mind](FORGEJO-SOCIETY-INTRODUCTION/THE-REPO-IS-THE-MIND/). Start here. |
| [FORGEJO-SOCIETY/](FORGEJO-SOCIETY/README.md) | The Forgejo-flavoured instance. Contains the only runnable subtree, [`forgejo-intelligence/`](FORGEJO-SOCIETY/forgejo-intelligence/README.md) — the Bun/TypeScript installer and Forgejo Actions workflows that bring an intelligence into a repo. |
| [FORGEJO-SOCIETY-PLAN/](FORGEJO-SOCIETY-PLAN/README.md) | Forward-looking design and planning: target layout, runtime pipeline, agencies, critics, censors, frames and K-lines, and the bootstrap checklist. |

| [FORGEJO-SOCIETY-THE-FEDERATION/](FORGEJO-SOCIETY-THE-FEDERATION/README.md) | The root society created by the project's maintainer. Federation-scope material, society minds, labour, and local hardware notes. |
