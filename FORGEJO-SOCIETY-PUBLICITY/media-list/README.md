# `media-list/`

> Outlets, journalists, and community publications we work with.

This folder holds the project's **media list**: the small set of
outlets, journalists, and community publications with which Forgejo
Society maintains a working relationship. Top-level rules live in
[`../03-media-relations.md`](../03-media-relations.md).

---

## What belongs here

- **`outlets.md`** — the canonical list of publications, with a short
  note on each one's audience and the kind of story it covers.
- **`relationships.md`** — a public-safe summary of working
  relationships: which outlets the project has spoken with, in what
  capacity, and when.
- **`inquiries.md`** — a log of substantive inquiries received, with
  status (acknowledged, declined, in progress, closed) and links to
  resulting coverage where applicable.

## What does not belong here

- **Personal contact details** of journalists or contributors. Those
  are kept out of the public repository.
- Speculative target lists for outreach the project has not actually
  done.
- Notes about peer projects' media relationships.

## Conventions

- **Public-safe by default.** Everything in this folder is written on
  the assumption it will be read by the people it describes.
- **Append-only inquiries.** The inquiries log is not rewritten;
  status changes are added as dated lines.
- **Small list.** We prefer a short list of meaningful relationships
  to a long list of weak ones.

## Status

Scaffold. The first entries are added when the project's first
inquiries arrive.
