---

**Forgejo Society** is a self-hosted cognitive forge built on
[Forgejo](https://forgejo.org/), running on Ubuntu hardware owned
and operated by the maintainers themselves.

It treats the forge — repositories, runners, issues, pull requests,
and CI/CD — not merely as a software platform, but as the operational
substrate for a governed society of agents.

Agencies. Critics. Censors. Memory.

A transparent and auditable cognitive ecology.

---

Forgejo Society makes five quiet reversals:

- The forge becomes the mind
- Intelligence becomes a governed society
- Capability is granted by files and audited by Git
- Cognition persists as Git objects
- Sovereignty becomes structural

Its implementation reflects this directly:

- one workflow file,
- one root folder,
- cognition reduced to durable primitives,
- runtime moved off hosted infrastructure.

In a world chasing demonstrations,
Forgejo Society pursues something rarer:

**durable sovereign intelligence.**
